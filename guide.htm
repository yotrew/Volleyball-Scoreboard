A. 網站管理者

  1. 上傳檔案
    1. 超連結為html檔

      2.   依照超連結檔名直接上傳同檔名即可,ex. "排球聯盟公定原則"所對應的檔名為"rule.htm"
        所以直接上傳名為"rule.htm"即可更新"排球聯盟公定原則"
    2. 超連結為php檔(程式)

      3.   必須所依照所規定檔名上傳(參照"相關檔案")

  2. 相關檔案
    1. 各隊資訊

      2.   各隊資訊有三個檔案 (a) teams.txt (b) teams_boy.txt (c) teams_girl.txt
        teams.txt的格式如下:
        (檔案第一行為"boy",第二行開始為男排各隊隊名,一行一隊,"girl"分隔男排和女排,檔案最後為[end])
        (menu會去讀teams.txt將各隊隊名列出來以供點選) 
      	  boy                   boy 
	  ...                   大機機隊
	  ...                   小機機隊
	  ...                   ...
	  girl                  girl
	  ...                   企管大女
	  ...                   企管小女
	  ...                   ...
	  [end]                 [end]

      teams_boy.txt和teams_girl.txt格式如下:
      (若沒有第二聯絡人或第三聯絡人則以空行)
      (隊員每行五個)
      (輸入完一隊後空一行再輸入下一隊)
      (沒有MSN帳號或電話則以"(空代替)" ex. 大空翼 (空) 091234567 xxx ) 
      	  (隊名)
	  (第一聯絡人)  (MSN帳號)  (電話)   (bala的ID)
	  (第二聯絡人)  (MSN帳號)  (電話)   (bala的ID)
	  (第三聯絡人)  (MSN帳號)  (電話)   (bala的ID)
	  [隊長]        (MSN帳號)  (電話)   (bala的ID)
	  (隊員名字) (隊員名字) (隊員名字) (隊員名字) (隊員名字)
	  (隊員名字) (隊員名字) (隊員名字) (隊員名字) (隊員名字)
	  ...
	  [end]
	  
	  {下一隊資訊}
	  ...

    2. 賽程
      3. 賽程檔案名稱格式為weekxx.htm,xx為週別
      ex. 第一週賽程檔案名稱為 week01.htm
      第二週賽程檔案名稱為 week02.htm
      以此類推.
      季後賽檔名為playoff.htm
    3. 組別
      4. 組別記錄檔為group.txt,一行輸入一組,前為組別名後為戰績記錄檔名
      	  ex. 男排2組 女排2組則輸入如下格式:
	  男排A組 boy_a
      男排B組 boy_b
      女排A組 girl_a
      女排B組 girl_b
  3. 戰績
    1. 戰績表新增
      2. 先選擇組別,然後輸入那一組的總隊數,再輸那一組的各隊名稱(以空格做區隔)
    2. 戰績(分數)新增/修改
      3. (依照網頁介面輸入即可,輸入分數較少一方,另一方分數會自動產生)

  4. Unicode問題

    5. 先使用"HTML文件外字相容轉換器"將字轉成代碼,然後以代碼取代字
    ex. 賴(方布)綺 -> 賴& #26046;綺 -> 賴斾綺
